# The Agentic Ecosystem Directory

This document is the **authoritative list of known AI harnesses, orchestrators, CLIs, and IDEs** for gitbox. It is embedded into the GUI binary via `//go:embed` and parsed at startup: rows whose `Executable / CLI Command` cell is a single backticked binary name (e.g. `` `claude` ``, `` `aider` ``) and whose `Category` is one of `Agentic CLI`, `AI Harness`, or `Headless Harness` are added to the "Open in AI harness" auto-detection list. Every other row is treated as reference-only documentation.

To add a new harness that gitbox should auto-detect, insert a new row with a single-identifier executable and one of the three eligible categories.
