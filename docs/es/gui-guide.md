# Guia GUI

[Read in English](../gui-guide.md)

## Primer uso

Abro GitboxApp, elijo la carpeta raiz y anado mis cuentas. La GUI usa la misma configuracion que la CLI.

## Idioma

En Ajustes puedo elegir `English` o `Espanol`. La seleccion se guarda en `global.language` y se aplica a los textos humanos que ya pasan por el traductor. Los datos de repos, cuentas, proveedores y estados internos permanecen sin traducir.

## Trabajo diario

- Uso la pestana Accounts para ver cuentas y clones.
- Uso Mirrors para copias entre proveedores.
- Uso Workspaces para abrir grupos de clones juntos.
- Uso el menu de cada repo para abrirlo en explorador, terminal, editor, navegador o harness de IA.
